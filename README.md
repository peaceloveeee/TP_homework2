# HW_2_TP

Проект по предмету **Технологии программирования**.

В проекте реализована простая система для работы с рецептами.  
Это консольное приложение, которое позволяет создавать блюда, добавлять их в "рецепты", масштабировать порции и генерировать список покупок.

- `recipes.py` — файл с основными классами:
  - `Ingredient`
  - `Recipe`
  - `ShoppingList`
  - `DietaryRecipe`
- `test_recipes.py` — тесты для проверки работы классов.
- `requirements.txt` — список зависимостей проекта.
- `.gitignore` — список файлов и папок, которые не нужно добавлять в Git.

## Установка

Чтобы запустить проект на новом компьютере, нужно склонировать репозиторий:

```bash
git clone https://github.com/peaceloveeee/TP_homework2.git
```

Перейти в папку проекта:

```bash
cd TP_homework2
```

Установить зависимости:

```bash
pip install -r requirements.txt
```

## Использование

Основной код находится в файле `recipes.py`.

Для запуска тестов нужно выполнить команду:

```bash
pytest
```

Если команда `pytest` не работает, можно запустить тесты так:

```bash
python -m pytest
```
